# Packages Needed for SRS PDF Generation

To generate the SRS PDF, the following packages are required:

1. **ReportLab** - A powerful library for creating PDF documents in Python.
2. **FPDF** - A simple PHP class for generating PDF files.
3. **PyPDF2** - A library for working with PDF files in Python.
4. **LaTeX** - A document preparation system for high-quality typesetting.

## Installation
Use the following commands to install the necessary packages:

```bash
pip install reportlab
pip install fpdf
pip install PyPDF2
```

For LaTeX, please follow the installation instructions based on your operating system.
